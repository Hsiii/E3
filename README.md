# EZE3 | E3 Login Automation

**EZE3** is a browser extension designed to eliminate friction when accessing E3, the NYCU learning platform. It automates the login process, bypass Google Authenticator 2FA, and provides seamless redirection to E3 from both the [NYCU Portal](https://portal.nycu.edu.tw/) and the [E3 login page](https://e3p.nycu.edu.tw/).

## Installation
The extension can be installed directly via [Chrome Web Store](https://chromewebstore.google.com/detail/ekijjgdmninecmfmlkaclcdfgbobaenc?utm_source=item-share-cb). 

After adding the extension to your browser, 
1. Visit [portal.nycu.edu.tw](https://portal.nycu.edu.tw/) and click the "Save to EZE3" button after entering user credentials or directly activating the extension to open a settings window to set up user credential auto-fill. 

2. After your first automated login with EZE3 (possible with manual 2FA input), you will be prompt to reset the 2FA settings and save the new Google Authenticator QR code into both EZE3 and your Google Authenticator app. Make sure you save it into EZE3 first!

3. You will be then redirected to E3 if everything went well. Congrats! From now on, no more manual logins and navigations before accessing E3 for you.
